## Introduction
Next.js is a React framework for production. It enables you to build full-stack web applications.

## Getting Started
```bash
npx create-next-app@latest
cd your-app-name
npm run dev
```

## Core Concepts
- **Pages**: File-based routing system
- **API Routes**: Serverless functions
- **Static Generation**: Pre-render at build time
- **Server-Side Rendering**: Render on each request

## Routing
Next.js uses file-based routing in the `app` or `pages` directory.

## Data Fetching
- `getStaticProps`: Static generation
- `getServerSideProps`: Server-side rendering
- `fetch()`: Built-in async/await support

## Styling
- CSS Modules
- Tailwind CSS
- CSS-in-JS libraries

## Deployment
Deploy to Vercel, Netlify, or any Node.js hosting platform.
